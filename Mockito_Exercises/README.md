# Mockito Hands-On Exercises

This repository contains the implementation of the Mockito Hands-On Exercises using **Java**, **Maven**, **JUnit 5**, and **Mockito**.

## Project Structure

### Source Files (`src/main/java/com/cts/mockito`)

* `ExternalApi.java`
* `MyService.java`
* `NotificationService.java`
* `NotificationManager.java`
* `DataService.java`
* `DataManager.java`
* `OrderService.java`
* `OrderManager.java`
* `PaymentService.java`
* `PaymentManager.java`

### Test Files (`src/test/java/com/cts/mockito`)

* `MyServiceTest.java` – Exercise 1: Mocking and Stubbing
* `MyServiceVerifyTest.java` – Exercise 2: Verifying Interactions
* `MyServiceArgumentTest.java` – Exercise 3: Argument Matching
* `NotificationManagerTest.java` – Exercise 4: Handling Void Methods
* `DataManagerTest.java` – Exercise 5: Mocking and Stubbing with Multiple Returns
* `OrderManagerTest.java` – Exercise 6: Verifying Interaction Order
* `PaymentManagerTest.java` – Exercise 7: Handling Void Methods with Exceptions

## Exercises Implemented

1. Mocking and Stubbing
2. Verifying Interactions
3. Argument Matching
4. Handling Void Methods
5. Mocking and Stubbing with Multiple Returns
6. Verifying Interaction Order
7. Handling Void Methods with Exceptions

## Technologies Used

* Java
* Maven
* JUnit 5
* Mockito
* Eclipse IDE

## How to Run

1. Import the project into Eclipse as a Maven project.
2. Update Maven dependencies.
3. Open any test class from `src/test/java/com/cts/mockito`.
4. Run the test using **Run As → JUnit Test**.
5. Verify that the test executes successfully (green JUnit bar).
