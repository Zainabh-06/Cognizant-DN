# Spring Data JPA Hands-on Exercises

## Objective
Learn and implement Spring Data JPA with Hibernate and MySQL by performing basic CRUD operations on the Country table.

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- VS Code

## Hands-ons Completed

### Hands-on 1
- Created a Spring Boot project using Spring Initializr.
- Configured MySQL database connection.
- Created `Country` entity, `CountryRepository`, and `CountryService`.
- Retrieved all countries from the database.

### Hands-on 4
- Studied the differences between JPA, Hibernate, and Spring Data JPA.
- Understood how Spring Data JPA simplifies database operations.

### Hands-on 5
- Populated the `country` table with sample country records.
- Verified the inserted data using SQL queries.

### Hands-on 6
- Implemented **Find Country by Code** using `findById()`.
- Created a custom `CountryNotFoundException`.
- Tested the functionality successfully.

### Hands-on 7
- Implemented **Add New Country** using `save()`.
- Inserted a new country record into the database and verified the result.

## Outcome
Successfully built a Spring Data JPA application connected to MySQL and implemented basic CRUD operations using Hibernate and Spring Data JPA.
