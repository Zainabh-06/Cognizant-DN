# Library Management - Spring Framework Hands-on

## Description

This project demonstrates the fundamentals of the Spring Framework through a simple Library Management application. It covers Spring Core, Dependency Injection (IoC), Spring AOP, Maven configuration, and Spring Boot REST API development.

## Technologies Used

* Java
* Spring Core
* Spring AOP
* Spring Boot
* Spring Data JPA
* H2 Database
* Maven
* VS Code

## Exercises Completed

* Exercise 1: Basic Spring Application Configuration
* Exercise 2: Dependency Injection (Setter Injection)
* Exercise 3: Logging with Spring AOP
* Exercise 4: Maven Project Configuration
* Exercise 5: Spring IoC Container Configuration
* Exercise 7: Constructor and Setter Injection
* Exercise 9: Spring Boot REST API with H2 Database

## How to Run

1. Open the project in VS Code.
2. Build the project using Maven.
3. Run the main application class (Spring Core) or the Spring Boot application.
4. Access the REST endpoints at `http://localhost:8080/books` for the Spring Boot application.
